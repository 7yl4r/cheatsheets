1. Create `.csv` tables.
2. Access an IPT (email a node manager).
3. Upload `.csv` files to IPT.
4. Complete metadata in the IPT.
5. Download DwC archive from IPT.
6. Check DwC archive with [BioCheck tool](https://rshiny.lifewatch.be/BioCheck/).
7. Set IPT dataset to public.
8. OBIS (and others) will harvest your data from the IPT.
